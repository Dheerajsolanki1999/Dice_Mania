# Dice Rolling Game

Welcome to the Dice Rolling Game! This is a web-based game where players roll dice, accumulate points, and aim to reach a score of 100 to win. The game is developed using HTML, CSS, and JavaScript.

#Play the game : https://dheerajsolanki1999.github.io/Dice_Mania/

## How to Play

1. Open the game in a web browser.
2. Player 1 starts the game.
3. Click on the "Roll" button to roll the dice. The dice animation will show the result.
4. If a player rolls a 1, their current score resets to 0, and it becomes the next player's turn.
5. If a player rolls any number other than 1, the number is added to their current score.
6. The player can choose to "Hold" their current score by clicking the "Hold" button. The current score is added to their total score, and it becomes the next player's turn.
7. The game continues with players taking turns until one player reaches a score of 100 or more.
8. The player who reaches the winning score first is declared the winner.

## Features

- Interactive dice rolling animation.
- Dynamic score tracking for each player.
- Visual indication of the current player.
- Reset of current score on rolling a 1.
- Victory celebration when a player reaches the winning score.

## Getting Started

To get started with the Dice Rolling Game, follow these steps:

1. Clone the repository to your local machine.
2. Open the `index.html` file in a web browser.
3. Play the game by rolling the dice and enjoy!
